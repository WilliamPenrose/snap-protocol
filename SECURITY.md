# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SNAP Protocol, **please do not open a public issue**.

Instead, use [GitHub's private vulnerability reporting](https://github.com/WilliamPenrose/snap-protocol/security/advisories/new) to report it confidentially.

We will acknowledge your report within 48 hours and work with you on a fix before any public disclosure.

## Scope

Security-relevant areas include:

- BIP-340 Schnorr signature signing and verification
- BIP-341 Taproot tweak implementation
- NIP-44 message encryption and decryption
- Message validation and replay protection
- Key management and identity handling

## Supported Versions

| Version | Supported |
|---------|-----------|
| SDK 0.4.x | Yes |
| < 0.4.0 | No |
