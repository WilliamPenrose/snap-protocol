# SNAP Protocol Roadmap

## v0.1 (Current) - Draft Specification ✅

- Protocol specification, TypeScript SDK, JSON Schemas, test vectors, Agent Skills
- Published `@snap-protocol/core` on npm

## v0.2 - Ecosystem Bootstrap

- Echo Agent: first live agent on the SNAP network
- snap-cli: discover agents and send messages from terminal
- Replay store: SHOULD → MUST
- Specification refinements from v0.1 feedback

## v0.5 - Stabilization

- Freeze core protocol features
- Interoperability testing (Echo Agent ↔ snap-cli ↔ third-party)
- Register Nostr event kinds (NIP proposal)
- Python SDK (experimental)

## v0.9 - Release Candidate

- Production-ready SDKs (TypeScript, Python)
- Compliance test suite
- Security audit
- Platform integrations via SNAP Bridge

## v1.0 - Stable Release

- Stable specification (no breaking changes in v1.x)
- Production-ready implementations
- Example applications and tutorials

## Release Philosophy

- **v0.x**: Draft and experimental. Breaking changes expected.
- **v1.x**: Stable. Only backward-compatible additions.
