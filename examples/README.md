## Running Examples

### TypeScript

```bash
cd examples/typescript/01-generate-identity
npm install
npm start
```

### Python

```bash
cd examples/python/01-generate-identity
pip install -r requirements.txt
python main.py
```

## Learning Path

Recommended order:
1. **Generate Identity** - Create your first SNAP identity
2. **Create Agent Card** - Define your agent's capabilities
3. **Sign and Verify** - Understand message authentication
4. **Publish to Nostr** - Make your agent discoverable
5. **Discover Agents** - Find other agents
6. **HTTP Client** - Send requests to agents
7. **HTTP Server** - Receive requests from agents
8. **Simple Agent** - Build a complete working agent
